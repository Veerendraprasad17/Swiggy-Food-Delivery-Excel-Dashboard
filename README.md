# Swiggy Food Delivery Excel Dashboard

## Project Overview
This project presents an interactive Excel dashboard created using Swiggy food delivery data. The dashboard helps analyze sales performance, restaurant trends, customer ratings, city-wise demand, food categories, and veg/non-veg distribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning
- Dashboard Design

## Key Insights
- Analyzed 197K+ food delivery records
- Tracked total sales, average rating, and order trends
- Identified top-performing restaurants and cities
- Compared veg and non-veg food performance
- Created interactive filters for better business analysis

## Dashboard Features
- Total Sales KPI
- Average Rating KPI
- City-wise Sales Analysis
- Restaurant Performance
- Food Category Analysis
- Veg vs Non-Veg Comparison
- Interactive Slicers

## Business Value
This dashboard helps business teams understand food delivery performance, identify high-demand areas, monitor restaurant ratings, and make better data-driven decisions.
